# 📋 数据完整性守则

## 🚫 **核心原则：绝不使用模拟数据**

### **第一条：数据来源必须100%真实**
- ✅ **允许**：QLIB真实市场数据
- ✅ **允许**：用户提供的CSV真实数据
- ✅ **允许**：公开市场API获取的真实数据
- ❌ **禁止**：`np.random.randn()` 生成模拟数据
- ❌ **禁止**：`np.random.uniform()` 生成模拟数据
- ❌ **禁止**：任何形式的随机数据生成
- ❌ **禁止**：预设相关性的模拟数据

### **第二条：数据泄漏严格禁止**
- ❌ **禁止**：因子生成时直接使用目标变量
- ❌ **禁止**：使用未来数据（look-ahead bias）
- ❌ **禁止**：在训练数据中混入测试数据
- ✅ **必须**：严格的时间序列分割
- ✅ **必须**：使用`shift(-1)`计算次日收益率

### **第三条：IC和ICIR必须真实**
- ✅ **必须**：基于真实市场数据计算IC
- ✅ **必须**：IC值范围应在真实市场范围（-0.05到0.05）
- ✅ **必须**：ICIR值范围应在真实市场范围（-1.0到1.0）
- ❌ **禁止**：IC值超过0.1（完全不现实）
- ❌ **禁止**：ICIR值超过2.0（完全不现实）

## 🔧 **技术实施规范**

### **数据加载规范**
```python
# ✅ 正确：使用QLIB真实数据
from qlib.data import D
df = D.features(['SH600000'], ['$close', '$volume'], start_time='2020-01-01', end_time='2021-12-31')

# ❌ 错误：使用模拟数据
import numpy as np
df['fake_factor'] = np.random.randn(len(df))  # 禁止！
```

### **因子计算规范**
```python
# ✅ 正确：基于真实价格计算技术指标
df['return_1d'] = df['close'].pct_change()
df['ma5'] = df['close'].rolling(5).mean()

# ❌ 错误：生成模拟因子
df['fake_factor'] = np.random.randn(len(df)) * 0.1  # 禁止！
```

### **目标变量规范**
```python
# ✅ 正确：计算次日收益率
df['target'] = df.groupby('code')['close'].shift(-1).pct_change()

# ❌ 错误：与因子相关生成目标
df['target'] = df['factor1'] * 0.1 + np.random.randn()  # 禁止！
```

## 🚨 **违规检测机制**

### **自动检测规则**
1. **IC值检测**：如果`abs(IC) > 0.1`，立即报警
2. **ICIR值检测**：如果`abs(ICIR) > 2.0`，立即报警
3. **数据来源检测**：检查是否使用`np.random`函数
4. **相关性检测**：检查因子与目标变量的预设相关性

### **代码审查要点**
```python
# 审查点1：检查import语句
import numpy as np  # ✅ 允许，但需审查使用方式

# 审查点2：检查数据生成
if 'np.random' in code:  # 🚨 需要详细审查
    raise ValueError("检测到模拟数据生成，违反守则")

# 审查点3：检查IC计算
```
